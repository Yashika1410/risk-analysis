Features

CRUD Operations on Each Table
Schedule Cron Jobs
Proper Logging
Get latest analysied data

Problem Statement

User have some raw data and wants to process it using some parameters and formulas with the help of a application, which process data using a scheduled job or it can triggered by the user.
The parameters and formulas can be changed because the processing of the data is under hit and trial process, so the raw data entities can be change as per requirement.

Assumptions made while building solution

  • The variable which are present in formula table should be present in weight and company risk score table/score table.
  • If any new dimension is added for a single company and that dimension is used in the formula table then we need to update all the remaining companies with that dimension using a default score which does not effect their analysis.

Limitations of the solution

  • Total risk capped score cannot be changed because it is hard coded in the code.
